# Problem statement

enables a google cloud platform service for a project

# Example usage

## Visualize

```shell
opctl ui github.com/opspec-pkgs/gcp.services.enable#1.0.0
```

## Run

```
opctl run github.com/opspec-pkgs/gcp.services.enable#1.0.0
```
